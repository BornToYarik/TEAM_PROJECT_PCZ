Opis szczegółowy

Kontekst React do zarzadzania lista zyczen (ulubionymi produktami).

Umozliwia dodawanie i usuwanie produktow z listy oraz ich trwale przechowywanie w pamieci lokalnej przegladarki (localStorage).
