Opis szczegółowy

Hub SignalR odpowiedzialny za komunikację w czasie rzeczywistym dla aukcji.

Umożliwia dołączanie użytkowników do aukcji, składanie ofert oraz przesyłanie aktualizacji ceny i czasu zakończenia aukcji do wszystkich uczestników.
