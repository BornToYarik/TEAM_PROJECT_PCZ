export const useAuth
Wartość początkowa:
= () => {
const token = localStorage.getItem("token");
const role = localStorage.getItem("role");
return {
isAuthenticated: !!token,
isAdmin: role === "Admin"
};
}

@description Funkcja pomocnicza (helper) do sprawdzania stanu autoryzacji uzytkownika.

Pobiera token oraz role z magazynu lokalnego (localStorage) w celu weryfikacji uprawnien.

Zwraca
{Object} Obiekt zawierajacy flagi logiczne: isAuthenticated (czy zalogowany) oraz isAdmin (czy posiada uprawnienia administratora).
