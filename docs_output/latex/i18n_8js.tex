\doxysubsection{Opis szczegółowy}
Modul konfiguracji tlumaczen i lokalizacji aplikacji Tech\+Store. 

Plik odpowiada za inicjalizacje biblioteki i18next, automatyczne wykrywanie jezyka uzytkownika oraz mapowanie plikow JSON z tlumaczeniami na wspierane kody jezykowe (PL, EN).
